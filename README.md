# Vigil Waitlist

Early access waitlist for Vigil - Revenue Protection SaaS

## Features

- Email signup with referral gamification
- Position tracking on waitlist
- Social sharing (Twitter, LinkedIn)
- Cost of downtime calculator
- Interactive pricing showcase
- FAQ section

## Setup

```bash
npm install
NEXT_PUBLIC_SUPABASE_URL=... NEXT_PUBLIC_SUPABASE_ANON_KEY=... npm run dev
```
